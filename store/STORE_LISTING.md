```
XP Explorer style bookmark manager with modern Liquid Glass theme. Features split view and drag-and-drop organization.
```

**Category:** Productivity

**Language:** English

---

## Detailed Description (for store listing)

```
Bookmark XP Explorer brings back the nostalgic XP Explorer interface to manage your Chrome bookmarks with a modern twist!

FEATURES:

★ Classic XP Look
- Familiar folder tree navigation
- Address bar with breadcrumb path
- Status bar with item count

★ Modern Liquid Glass Theme
- Clean, translucent design
- Smooth animations and transitions
- Beautiful icon set

★ Split View Mode
- View two folders side-by-side
- Drag and drop between panes
- Perfect for organizing bookmarks

★ Full Bookmark Management
- Create, rename, and delete bookmarks/folders
- Drag and drop to reorganize
- Search across all bookmarks
- Keyboard shortcuts support

★ Dual View Options
- Compact popup for quick access
- Full-page mode for power users
- Toggle with Ctrl+Shift+B (Cmd+Shift+B on Mac)

★ Privacy Focused
- No data collection
- No external connections
- All data stays in your browser

Perfect for users who miss the classic file explorer experience and want a powerful way to organize their bookmarks!
```

```
This extension manages Chrome bookmarks using the chrome.bookmarks API.

Permissions justification:
- bookmarks: Core functionality - reading and modifying user's bookmarks
- storage: Saving user preferences (view mode, expanded folders)
```
